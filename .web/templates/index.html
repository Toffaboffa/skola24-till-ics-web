Skola24 till iCal (.ics)

Av Kristoffer Åberg

Denna webbplats hjälper dig att generera en iCal (.ics) fil för ditt schema, för import till din kalender. Fyll i formuläret nedan med information om din stads domän, skolans namn och ID, samt årets ID. Tryck sedan på "Generera .ics" för att skapa filen. iCalendar användas i din kalenderapp, Outlook eller annat program som stöder iCal-formatet.

Exempel: gotham.skola24.se Exempel: Gotham High Exempel: ODBmYWIzYjotM2Yw... (48 tecken) Exempel: 7d35b17b-42eb-46... (32 tecken) Exempel: BW
